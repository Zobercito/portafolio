# Utilities Directory

This directory contains standalone helper functions and API clients.

## 🔧 Utility Tools

- **`groqClient.ts`**: Configuration and client for the GROQ API (used by the MiniChat).
- **`modalUtils.ts`**: Helpers for managing modal states, scroll locking, and accessibility.

## 🛠️ Guidelines
Keep these functions "pure" and environment-agnostic where possible, making them easy to test and reuse.
