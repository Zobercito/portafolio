# TypeScript Definitions (Types)

This directory centralizes all the Type and Interface definitions used across the project to ensure total type safety.

## 📂 Type Modules

- **`portfolio.ts`**: Interfaces for the data stored in `src/data/` (Project, Experience, Certification).
- **`chat.ts`**: Types for the AI MiniChat (Message roles, Chat history, API responses).

---
> [!NOTE]
> Centralizing types avoids circular dependencies and makes the codebase much easier to refactor.
